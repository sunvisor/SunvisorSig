# MVP 画面一覧

## 前提

- 認証は初期実装の対象外とする
- まずは開発用として、ユーザーは既存データとして扱う
- 最初の実装対象はフォーラム、チャンネル、投稿、コメント、添付ファイルとする

## 画面一覧

### 1. フォーラム一覧

- URL: `/forums`
- 目的: 参加可能なフォーラムを一覧表示する
- 主な要素:
  - フォーラム名
  - フォーラム説明
  - フォーラムへの導線

### 2. フォーラム詳細

- URL: `/forums/[forumId]`
- 目的: フォーラム内のチャンネル一覧を表示する
- 主な要素:
  - フォーラム名
  - フォーラム説明
  - チャンネル一覧
  - チャンネル作成導線

### 3. チャンネル詳細

- URL: `/forums/[forumId]/channels/[channelId]`
- 目的: チャンネル内の投稿一覧を表示する
- 主な要素:
  - チャンネル名
  - チャンネル説明
  - 投稿一覧
  - 投稿作成導線

### 4. 投稿作成

- URL: `/forums/[forumId]/channels/[channelId]/posts/new`
- 目的: 新規投稿を作成する
- 主な要素:
  - タイトル入力
  - Markdown 本文入力
  - 添付ファイル追加
  - 投稿作成ボタン

### 5. 投稿詳細

- URL: `/forums/[forumId]/channels/[channelId]/posts/[postId]`
- 目的: 投稿本文とコメント一覧を表示する
- 主な要素:
  - 投稿タイトル
  - Markdown 本文
  - 添付ファイル一覧
  - コメント一覧
  - コメント投稿フォーム
  - 投稿削除導線

### 6. コメント投稿

- URL: 投稿詳細画面内
- 目的: 投稿へのコメントを追加する
- 主な要素:
  - Markdown 本文入力
  - 添付ファイル追加
  - コメント投稿ボタン

### 7. チャンネル作成

- URL: `/forums/[forumId]/channels/new`
- 目的: フォーラム内にチャンネルを作成する
- 主な要素:
  - チャンネル名入力
  - 説明入力
  - 作成ボタン

## 初期実装で含めない画面

- ログイン画面
- アクティベーション画面
- 招待管理画面
- 参加者管理画面
- 削除履歴画面
- 添付ファイル単体管理画面

## 補足

- コメントへのコメントは作らない
- 投稿とコメントの本文は Markdown で保存する
- 添付ファイル参照は [markdown.md](/Users/sunvisor/projects/SunvisorSig/specs/markdown.md) の独自記法に従う
