### 目的:
顧客と私がやり取りできるフォーラムのシステムを作ります
### 登場人物
    - 管理者: フォーラム全体を運営する人
    - 利用者: フォーラムに参加する人（管理者も参加できる）
### 仕様:
- フォーラム - チャンネル - 投稿 - コメント といった構造
- フォーラム
    - 複数作成可能
    - 作成は管理者
    - 参加者に対して招待して参加してもらう
    - 顧客ごとにフォーラムを作成する
    - チャンネルを複数設定できる
- チャンネル
    - フォーラム内に複数作成可能
    - 作成は管理者
    - 新しく投稿を作成できる（利用者）
    - 投稿を削除できる（投稿した人）
- 投稿
    - Markdown で記述できる
    - 画像・動画・資料などのファイルを添付できる
    - 投稿についてコメントできる（利用者）
- コメント
    - 投稿に対してコメントできる（参加者）
    - 画像・動画・資料などのファイルを添付できる
    - コメントを削除できる（コメントした人）
    - コメントに対してのコメントはできない
- 認証
    - 招待のメールにアクティベーションページを送る
    - アクティベーションページで利用者がプロフィールとパスワードを設定する
    - ログインして利用可能
### 技術選択
- フレームワーク: Next.js
- UI: Tailwind CSS
- データベース: PostgreSQL
- ORM: Prisma
- 開発環境: Docker
